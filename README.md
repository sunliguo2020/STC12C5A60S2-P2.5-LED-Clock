# STC12C5A60S2 驱动 75 接口点 P2.5 全彩屏 万年历时钟

本项目是基于 STC12C5A60S2 单片机的 P2.5 全彩点阵时钟程序，支持 DS3231 RTC、GPS 时间校准、DS18B20/NTC 温度检测、红外遥控学习和按键设置。项目主要源代码位于 `3264_DS3231_摇控_P2.5RGB时钟/` 目录。

## 功能概述

- 基于 STC12C5A60S2 MCU 的多功能全彩时钟
- 使用 DS3231 RTC 实现精准时钟和日历
- 支持 GPS 模块 GPRMC 数据接收与校时
- 支持 DS18B20 温度传感器，兼容 NTC 温度检测模式
- 支持 75 接口点 P2.5 全彩 LED 模块显示
- 支持红外遥控学习和自定义按键控制
- 支持多种显示模式、颜色模式和亮度调节
- 参数保存在片内 EEPROM，断电后可保持设置

## 主要硬件接口

- MCU: STC12C5A60S2
- 实时时钟芯片: DS3231
- 温度传感器: DS18B20 或 NTC
- GPS: 串口接收 GPRMC 数据
- LED 显示: P2.5 全彩点阵模块，75 接口驱动
- 红外接收: 支持代码学习与遥控按键
- 本地按键: SET、+、-、ESC

## 目录结构

- `3264_DS3231_摇控_P2.5RGB时钟/` - 主程序目录
  - `3264gps时钟.c` - 主程序入口、初始化、定时扫描与主循环
  - `DS3231+AT24C32.h` - DS3231 RTC 与 I2C 通讯、时间读取/写入
  - `18b20.h` - DS18B20 温度读取模块
  - `set.h` - 显示模式与设置界面逻辑
  - `eep.h` - 片内 EEPROM 读写、参数保存、红外码存储

  - `.hex` / `.obj` / `.lnp` / `.M51` - 编译生成文件

## 运行说明

1. 使用 Keil μVision 或兼容 STC C51 编译器打开 `3264_DS3231_摇控_P2.5RGB时钟/3264GPS时钟-12.uvproj` 或 `3264GPS时钟.uvproj`。
2. 编译生成 `.hex` 文件后，用 STC-ISP 或兼容下载工具烧录到 STC12C5A60S2 芯片。
3. 上电后程序初始化 DS3231、ADC、串口、定时器、中断和显示驱动。
4. 如果首次运行或 EEPROM 未保存参数，将写入默认参数并显示初始化画面。
5. 通过按键或红外遥控进入设置模式，调整显示模式、时间、亮度、色彩等。

## 操作说明

- `SET`：进入/退出设置或切换显示模式
- `+`：调整当前选项的增加值或切换模式
- `-`：调整当前选项的减少值
- `ESC`：取消设置、退出并返回正常显示

红外遥控支持学习功能，程序会读取并保存 5 个红外按键代码到片内 EEPROM。具体按键码与红外遥控器类型需要根据实际硬件学习结果匹配。

## 显示与模式

程序内置多种显示模式，常见模式包括：

- 实时时钟显示
- 时间与日期显示
- 星期显示
- 农历/节气显示
- 彩色效果显示
- 温度显示

颜色模式通过 `COLORMODE` 控制，包含单色、RGB 过渡和自动变色等效果。亮度通过 ADC 自动或手动调节。

## 重要提醒

- 代码中使用 STC12C5A60S2 专用寄存器与 IAP 片内 EEPROM 操作，编译时请使用兼容 STC 的 C51 编译器。
- GPS 校时功能依赖串口接收 `GPRMC` 数据帧，若未接入 GPS 模块则默认关闭 GPS 功能。
- DS3231 RTC 与 RGB 显示电路应正确接地，并保障供电稳定。
